документация для библиотеки watchdog timer qt

как заставить работать:

1. установить драйвер согласно README из CFILES

2. поместить библиотеку CFILES/lib/x64 в usr/lib

3. поместить headers из CFILES/include в usr/include

4. поместить headers из sourceCode в usr/include

5. поместить .so файл из libFile в usr/lib

6. добавить библиотеку из предыдущего в проект qt


небольшое описание:

	когда создаётся объект класса WatchdogTimer, с параметрами (int timeCheck/* время проверки работы приложения и обновления wdt */, int timeWdt /* время таймаута wdt , если его не сбрасывать */)
	, одновременно запускаются два таймера watchdog timer и обычный таймер. wdt установлен на timerWdt сек и перезагрузит систему, если досчитает до конца, а обычный таймер сбрасывает его каждые
		timeCheck сек. Желательно объявлять объект WatchdogTimer динамически и ОБЯЗАТЕЛЬНО уничтожить его в деструкторе класса где вы его вызвали.
